# Clover 반복 타이머

38초, 50초, 60초, 100초를 선택해 반복 실행하는 터치용 타이머입니다.

## 동작

- 메인 화면에서 `38초`, `50초`, `60초`, `100초` 중 하나를 터치하면 시작
- 선택한 시간으로 계속 반복
- 마지막 3초는 화면이 반짝임
- 상단 `MAIN` 버튼을 누르면 메인 화면으로 복귀
- 타이머 화면의 다른 곳을 누르면 처음부터 다시 시작

## 스마트폰 사용

- Android: Chrome에서 열고 `앱 설치` 또는 `홈 화면에 추가`
- iPhone: Safari에서 열고 공유 버튼 → `홈 화면에 추가`
