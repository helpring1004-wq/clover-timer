# 크로버 반복 타이머 설치 방법

## Android

1. Chrome으로 `index.html`을 연다.
2. 주소창 메뉴에서 `홈 화면에 추가` 또는 `앱 설치`를 선택한다.
3. 설치 후 아이콘을 눌러 독립 앱처럼 실행한다.

## iPhone

1. Safari로 `index.html`을 연다.
2. 공유 버튼을 누른다.
3. `홈 화면에 추가`를 선택한다.
4. 홈 화면 아이콘으로 실행한다.

## 참고

- 인터넷 없이도 열린 적이 있으면 동작할 수 있게 `service worker`를 넣어뒀다.
- 로컬 파일로만 열면 브라우저에 따라 설치 옵션이 제한될 수 있으니, GitHub Pages 같은 HTTPS 환경에 올리는 것을 권장한다.
